# Simple Calculator

A basic **calculator web app** built with **HTML, CSS, and JavaScript**.  
It supports addition, subtraction, multiplication, and division.

## Features

- Perform basic arithmetic operations: `+`, `-`, `*`, `/`
- Decimal support
- Clear (`C`) button to reset input
- Responsive layout for desktop and mobile
- Simple, clean design

## How to Run

### Option 1: Open Locally
1. Clone the repo or download the files.
2. Open `index.html` in any web browser.
3. Use the calculator!

### Option 2: Using VS Code Live Server (Optional)
1. Open the project in VS Code.
2. Install the **Live Server** extension.
3. Right-click `index.html` → **Open with Live Server**.
4. Your browser will open with live reload support.

## File Structure
1. index.html # Main HTML file
2. style.css # Styles for calculator
3. script.js # Calculator logic
